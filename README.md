## 설명
스크롤 없이 한눈에 딱 필요한 정보만 제공하는 간단한 날씨 앱.

어제와의 기온 비교를 통해 체감의 기준을 가진다. 

미세먼지는 현재 위치에 근접한 측정소에서 제공하는 수치를 국제 기준(WHO)으로 제공한다.


본 앱은 기상청, 한국환경공단, kakao open api를 이용하였고, 아이콘은 skplanet 제공 아이콘을 사용하였습니다.
